What is LAMP ?

  • L - Linux (OS) eg:RedHat, Ubuntu, CentOS
  • A - Apache (Web Server)
  • M - MySQL (Database Server)
  • P - PHP / Python / Perl / PgSQL
LAMP is widely used over the world wide web.

Why we use it ?

Open Source

More Secure

Low Costs

Great Performance

Highly Scalable

and many more...
